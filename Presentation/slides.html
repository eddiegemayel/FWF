What's Meteor?


  • A javascript framework
  • Installed via Terminal Command ; no download available
  • Runs off of their server (localhost:3000)

Getting Set up via Terminal Commands


Step 1: Install it

							
$ curl https://install.meteor.com | /bin/sh

Step 2: Create Your Project

							
$ meteor create myapp

Step 3: Run It Locally

							
$ cd myapp

$ meteor

Step 4: (Optional) Deploy

							
$ meteor deploy myapp.meteor.com
